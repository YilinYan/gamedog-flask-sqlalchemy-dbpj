## manual
##### for macOS user
- install search enginee
```
$ brew install elasticsearch
```
- start enginee
```
$ usr/local/bin/elasticsearch &
```
- use environment
```
$ python3 -m venv venv
$ pip install ...
$ ...
$ source venv/bin/activate
```
- export env variable
```
$ export FLASK_APP=project.py
```
- start app
```
$ flask run
```
- go to "localhost:5000"
