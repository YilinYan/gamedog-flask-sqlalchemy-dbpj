## Get Started
#### For macOS
Install search enginee
```
$ brew install elasticsearch
```
Start the enginee
```
$ usr/local/bin/elasticsearch &
```
Set the environment
```
$ python3 -m venv venv
$ pip install ...
$ ...
$ source venv/bin/activate
```
Export env variable
```
$ export FLASK_APP=project.py
```
Start app
```
$ flask run
```
Go to "localhost:5000"
